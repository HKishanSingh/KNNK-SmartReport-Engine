# KNNK-SmartReport-Engine
KNNK SmartReport Engine automates GAM &amp; DCM reporting with campaign mapping, reconciliation, performance insights, and ML-based forecasting. Built with Streamlit, Python, XGBoost, Prophet, and Google Sheets integration to transform raw campaign data into actionable insights.

**🚀 Key Features**

**Automated Reporting**
* Upload GAM and DCM reports directly.
* Generate clean performance summaries instantly.
* Export processed reports to Excel.

  
**Smart Campaign Mapping**
* Dynamic keyword-based mapping engine.
* Google Sheets integration for centralized mapping management.
* Supports custom campaign configurations.


**Performance Insights Engine**
* Identifies top and low-performing placements.
* Highlights CTR trends, impression concentration, and optimization opportunities.
* Generates actionable recommendations automatically.


**Forecasting & Predictive Analytics**
* XGBoost-based forecasting model optimized for campaign delivery data.
* Prophet forecasting for long-duration campaigns with seasonal patterns.
* Confidence scoring and historical accuracy measurement.
* Automated holdout testing using MAPE-based accuracy calculations.

**GAM vs DCM Reconciliation**
* Compare delivery metrics across platforms.
* Detect discrepancies automatically.
* Flag reporting mismatches for investigation.

**Data Quality Validation**
* Historical data sufficiency checks.
* Forecast reliability assessment.
* Automated warning and confidence indicators.

**🛠 Technology Stack**
* Frontend/UI: Streamlit
* Data Processing: Pandas, NumPy
* Visualization: Streamlit Charts
* Machin learning: XGboost, Prophet
* Storage: Google Sheets API (gspread)
* Authentication: Google Service Account

**🎯 Business Value**

KNNK SmartReport Engine reduces manual reporting effort, improves forecast reliability, increases reporting accuracy, and provides a centralized analytics solution for campaign managers, ad operations teams, and digital marketing professionals.

**Forecast. Reconcile. Analyze. Automate. 🚀📈**
